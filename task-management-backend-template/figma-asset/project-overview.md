Project Overview

    Your system is a multi-role task management platform with three distinct user interfaces:

    ---

    1. Main Admin Dashboard (Web)
    For system administrators managing the entire platform:
     - Dashboard: Analytics with user stats, monthly income, user ratio charts
     - User List: Manage all users (Individual/Business), search/filter, view user details
     - Subscription: Create/manage subscription plans (Individual $10.99/mo, Group Plan $29.99/mo)
     - Settings: System configuration

    ---

    2. Teacher/Parent Dashboard (Web)
    For managing family/team members (Group Plan subscribers):
     - Dashboard: Team overview with member cards, task summaries, live activity feed, quick assign
     - Task Monitoring: Track tasks by status (Not Started, In Progress, Completed, My Tasks), activity charts
     - Create Task: Three types:
       - Single Assignment: Assign to one member
       - Collaborative Task: Assign to multiple members
       - Personal Task: For yourself
     - Team Members: Add/edit/remove members, view member details and their tasks
     - Subscription: View/manage plan (up to 5 users: 1 Primary + 4 Secondary)
     - Settings/Permissions: Control which secondary users can create/assign tasks

    ---

    3. App User (Mobile) - Children/Group Members
    For individual team members (secondary users):
     - Home: Task list with daily progress, support mode selection (Calm/Encouraging/Logical)
     - Add Task: Create tasks (permission-based)
       - With permission: Can create Single/Collaborative tasks for others
       - Without permission: Only Personal tasks for self
     - Status: View tasks by status (Pending, In Progress, Completed), subtask tracking
     - Profile: Personal info, support mode, notification style preferences

    ---

    Key Features Identified:
     - Support Modes: Calm, Encouraging, Logical (different motivational styles)
     - Subtasks: Tasks can have multiple subtasks with progress tracking
     - Real-time Activity: Live feed of team member completions
     - Permission System: Secondary users can be granted task creation rights
     - Multi-platform: Web dashboard + Mobile app
