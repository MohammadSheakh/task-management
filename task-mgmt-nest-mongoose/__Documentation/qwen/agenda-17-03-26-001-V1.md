## 📝 **Express → NestJS Transition Notes**

```typescript
// 📚 EXPRESS → NESTJS

// Express Pattern:
// - Manual middleware: auth(), rateLimiter()
// - Direct service calls: AuthService.login()
// - Manual error handling: try-catch

// NestJS Pattern:
// - @UseGuards(AuthGuard, ThrottleGuard)
// - DI: constructor(private authService: AuthService)
// - Exception filters handle errors globally

// Key Learnings:
// 1. Decorators replace middleware chains
// 2. DI eliminates manual instantiation
// 3. DTOs provide automatic validation
// 4. Redis is perfect for short-term data (OTP)

// Gotchas:
// - res.cookie() needs @Res({ passthrough: true })
// - req.user → @User() decorator
// - OTP TTL must be in seconds (not milliseconds)
```
